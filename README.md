## ecommerce

This is an API build using `node.js` and `mongodb`
follow this step
step 1: clone this app
step 2: install dependensi bellow

```javascript
"dependencies": {
"crypto-js": "^4.1.1",
"dotenv": "^16.0.2",
"express": "^4.17.1",
"jsonwebtoken": "^8.5.1",
"mongoose": "^6.6.2",
"nodemon": "^2.0.20"
}
```

step 3: create account in mongodb
step 4: install postman
step 5: run this app using vsCode npm
